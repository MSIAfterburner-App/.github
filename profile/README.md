## 🚀 Overview
MSI Afterburner for Windows is a free graphics card tool that combines detailed monitoring with hands-on control. Compatible with GPUs from any brand — not only MSI — it lets you adjust core and memory clocks, power limits, and fan curves to get the performance and temperatures you're after. A live monitor reports GPU load, clock speeds, and CPU temp, while the customizable overlay drops an FPS counter and hardware stats straight into your games. One MSI Afterburner download gives gamers and PC builders a single, dependable place to watch, tune, and protect their hardware.

2. Enable the overlay so your FPS counter and temperatures show up inside your games.
3. Adjust the core clock, memory clock, or fan curve gradually, applying each change as you go.

**What is MSI Afterburner?**
MSI Afterburner is a free graphics card utility for Windows used to monitor hardware, tune GPU performance, control fan speeds, and display an in-game overlay. It works with graphics cards from most manufacturers, not just MSI.

**How do I use MSI Afterburner?**
Start the app to see your GPU's live stats, then use the sliders to modify clock speeds, power limits, and fan curves. Apply the changes, keep an eye on CPU and GPU temps, and save the setup you prefer as a profile. Turn on the overlay to watch FPS and temperatures while gaming.

**Is MSI Afterburner free?**
Yes. MSI Afterburner is free to download and use, with no subscription required.

**Which versions of Windows are supported?**
